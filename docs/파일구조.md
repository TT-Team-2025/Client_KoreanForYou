## 🎯 프로젝트 개요

Front_KoreanForYou는 요식업 외국인 근로자들이 실무에서 필요한 한국어를 효과적으로 학습할 수 있도록 돕는 웹 애플리케이션입니다.

**핵심 기능**:
- 📚 레벨별 문장 학습 (초급/중급/고급)
- 🎤 발음 연습 및 피드백
- 🤖 AI 기반 대화 시뮬레이션
- 📊 학습 진도 추적 및 통계
- 👥 커뮤니티 기능 (게시판, 댓글)
